
### How to Use This

#### For learning (30-day roadmap)

1. Open `APRENDIZAJE/04-roadmap-aprendizaje.md`
2. Follow 15 minutes/day

#### To build your e-commerce

1. Open `00-prompt-inicio-deepseek.md`
2. Copy its content and paste it to **DeepSeek AI**
3. Follow the execution order in `PROMPTS ESPECÍFICOS/00-README.md`
4. The AI will generate ready-to-use code (SQL, TypeScript, Edge Functions)

### Key Prompts

| # | Prompt | What it generates |
|---|--------|-------------------|
| 12 | Products + Inventory | `products` table with `decrement_stock` |
| 13 | Stripe Webhook | Idempotency, rollback, stock control |
| 14 | Transactional Emails | Resend + React Email (RDL 1/2007) |
| 17 | Legal Invoicing | Invoice generation, sequential numbering |
| 21 | Returns + Refunds | 14-day withdrawal, Stripe refunds |
| 22 | Integration Tests | RLS, checkout, webhook tests |
| 23 | GDPR Retention | Automated data retention job |

### Requirements

- Supabase account (free tier works)
- Stripe account (test mode available)
- DeepSeek AI (free) or any LLM
- Node.js 18+ (for generated frontend code)

### What This Is NOT

- ❌ Not executable code (prompts are instructions for AI)
- ❌ Not a copy-paste ready project
