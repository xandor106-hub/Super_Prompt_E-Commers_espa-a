## English Version

### What is this?

This is a **professional prompt library** of **45+ files** to build a complete e-commerce in Spain with:

| Component | Technology |
|-----------|------------|
| **Backend** | Supabase (PostgreSQL, RLS, Edge Functions) |
| **Frontend** | Next.js 14 (App Router, TypeScript) |
| **Payments** | Stripe (3D Secure, webhooks, refunds) |
| **Legal Compliance** | GDPR, LSSI, LOPDGDD, ePrivacy, RDL 1/2007 |
| **Security** | Rate limiting, idempotency, real security tools |
| **Testing** | Integration tests (RLS, checkout, webhooks) |
