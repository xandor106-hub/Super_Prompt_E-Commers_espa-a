## 📋 ¿Qué es esto?

Una **biblioteca profesional de prompts** para construir un e-commerce completo en España, con:

- Backend seguro (Supabase, RLS, Edge Functions)
- Frontend moderno (Next.js 14, arquitectura por features)
- Cumplimiento legal (RGPD, LSSI, facturación, devoluciones)
- Integración robusta (tipos compartidos, transformadores)
- Operaciones (observabilidad, tests, retención de datos)

## 🚀 Cómo usar este tesoro

### Paso 1: Empieza con el aprendizaje (si eres nuevo)

1. Abre `📂 APRENDIZAJE/04-roadmap-aprendizaje.md`
2. Sigue el plan de 30 días (15 min/día)

### Paso 2: Cuando tengas el backend real (Datos Superbase o cual tengas)

1. Abre `00-prompt-inicio-deepseek.md`
2. Copia y pega su contenido en DeepSeek
3. Sigue el orden de `📜 PROMPTS ESPECÍFICOS/00-README.md`

### Paso 3: Construye el frontend

1. Sigue el roadmap visual en `📂 FRONTEND/Resumen_visual_roadmap.md`
2. Ejecuta los prompts en orden (Fase 1 → Fase 6)

### Paso 4: Integra todo

1. Ejecuta los prompts de `📂 INTEGRACION/` (INT-01, INT-02, INT-03)
2. Conecta backend con frontend

### Paso 5: Lanza a producción

1. Revisa `📂 APRENDIZAJE/05-checklist-seguridad.md`
2. Ejecuta `📜 PROMPTS ESPECÍFICOS/16-prompt-seguridad-preproduccion.md`
3. Despliega en Vercel (EU region)
